# Constitución Técnica (Resumen Operativo)

Reglas inmutables del MVP:

1) Event Sourcing append-only (prohibido UPDATE/DELETE en eventos).
2) No custodia de dinero, no integraciones bancarias.
3) No envío automático a autoridades.
4) No identidad civil almacenada (CURP/RFC/NSS/pasaporte/biometría prohibidos).
5) Separación absoluta identidad ↔ voto.
6) Anti-admin: no existe superusuario.
7) TrustLayer Nivel 3 existe estructuralmente pero permanece inactivo.
8) IA solo asistencial con grounding estricto y citación.
9) Evidencia nunca se borra; solo se restringe acceso por eventos.
10) Prohibido inventar features no especificadas.
